# Al-Sakhra-Educational-Academy Backend

Laravel API backend for Al-Sakhra Educational Academy.

## Stack

- Laravel 10
- MySQL
- Laravel Sanctum
- Spatie Laravel Permission

## Modules

- Multi-tenant organizations
- Users, roles, and permissions
- Registration codes and QR codes
- Grades, subjects, students, and teachers
- Courses, lessons, videos, and lesson files
- Contracts, commissions, payments, and wallets
- Exams, content protection, and notifications
